使用flexbox，创建一个3列布局，其中每列占据容器的“col - {n} / 12”比率

<div class="row">
  <div class="col-2"></div>
  <div class="col-7"></div>
  <div class="col-3"></div>
</div>

参考答案

将 .row 父项设置为 display：flex; 并使用 flex 简写属性为列类提供与其比值相对应的 flex-grow 值。

.row {
  display: flex;
}

.col-2 {
  flex: 2;
}

.col-7 {
  flex: 7;
}

.col-3 {
  flex: 3;
}
