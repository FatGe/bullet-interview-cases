JavaScript 有哪些数据类型

参考答案

最新的ECMAScript标准定义了七种数据类型，其中六种是简单数据类型（也称基本数据类型）：Boolean，Null，Undefined，Number，String，Symbol 和一种非原始数据类型：Object。

关键点

  • 提到新添加的 Symbol 数据类型;
  • Array，Date 和 function 都是 object 类型;
  • JavaScript中的函数是具有可调用功能的对象。
